1) Создать 2 сервисных класса Service1 и Service2
причем в Service1 должен быть метод String serviceMethod1(String arg1), а Service2 должен быть
метод String serviceMethod1(String arg1, String arg2, String arg3)

2) Создать класс, который бы в цикле вычитывал из входящего файла, заданного в аргументах
запуска программы класс сервиса, имя сервисного метода и аргументы, а затем запускал бы
указанный метод и результат запуска прописывал бы в исходящий файл. Причем создание
экземпляра сервиса и вызов метода должны происходить через reflection. Поиск метода должен
происходить по методу и аргументам, указанным в файле. Запись в файле может иметь вид
package.SeviceClass serviceMethod arg1 arg2 arg3
Тип аргументов считать String.

3) Добавить вывод через логгер:
а. Ошибки возникающие при неправильном задании класса, метода или количества аргументов
для вызова сервиса - в уровень ошибок
б. выводить в инфо информацию о старте обработки строки из файла и об окончании обработки
строки.
в. При каждом вызове сервиса выводить имя сервиса, метод, входящие параметры, результат
выполнения сервиса в дебаг
